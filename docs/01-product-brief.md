# Product Brief: Tukang-On

## Vision
Menjadi platform penghubung antara masyarakat dengan tukang servis terpercaya.

## Target Pengguna
- Pengguna jasa (user) - Mencari dan memesan tukang
- Penyedia jasa (tukang) - Menerima pesanan
- Admin - Mengelola platform

## Fitur Utama
1. Pencarian tukang berdasarkan keahlian dan lokasi
2. Pemesanan jasa dengan tracking status
3. Rating dan review
4. Notifikasi WhatsApp
5. Dashboard admin
